Track information.

This structure provides information about a movie track.

uint OmTrackInfo::compressedAudioChannelCount

Compressed audio channel count, or 0 if not applicable to this track.

For example, an AC-3 track with 5.1 audio will report 6 in this field.

char OmTrackInfo::compressedAudioFormat[32]

Compressed audio format per SMPTE 337 specification, if known.

Empty string for non-audio tracks, tracks that have PCM audio or tracks with an unknown compression format. Currently supported SMPTE 337 compressed audio formats are "AC-3" and "Dolby-E". Set to an empty string when writing a PCM track.

uint OmTrackInfo::decodeDelay

Latency in frames from start of decode to display of first frame.

For essence types that don't have their frames reordered this is typically 0. For MPEG-2 long-GOP the delay is 1 frame (i.e. the first picture will be displayed when the second frame is sent to the decoder). Some AVC long-GOP tracks can have longer delays.
