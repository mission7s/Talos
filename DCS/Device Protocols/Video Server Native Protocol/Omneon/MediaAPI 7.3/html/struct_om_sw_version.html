unsigned int OmSwVersion::getScalar ( ) const [inline]

Return a version number encoded as a single integer.

Returns:
An 32-bit integer that encodes the four components of a version number.
Remarks:
The major, minor, service and hotfix numbers each use 8-bits. The 8 most significant bits are the major number, followed by minor, service and hotfix.
